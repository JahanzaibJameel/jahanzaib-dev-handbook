## 🎯 About This Project

The **Jahanzaib Dev Handbook** is a comprehensive, production-ready developer guide that teaches modern development from beginner to advanced levels. This isn't just a tutorial collection - it's a complete learning system that demonstrates real-world engineering skills, portfolio-worthy projects, and career development strategies.

### What Makes This Special

- **📚 Complete Learning Path**: 12-month structured curriculum from basics to advanced
- **🛠️ Real Projects**: Production-ready applications with full implementations
- **🤖 AI Integration**: Modern ChatGPT and AI development techniques
- **📱 Cross-Platform**: React Native mobile apps + Next.js web applications
- **💼 Career Focus**: Freelancing, branding, and interview preparation
- **🌍 Live Website**: Deployed and accessible to showcase your skills

## 🚀 Quick Start

### View the Live Handbook

👉 **[Live Demo](https://jahanzaibjameel.github.io/jahanzaib-dev-handbook)**

### Local Development

1. **Clone the repository**

   ```bash
   git clone https://github.com/JahanzaibJameel/jahanzaib-dev-handbook.git
   cd jahanzaib-dev-handbook
   ```

2. **Install mdBook**

   ```bash
   # Install Rust first, then:
   cargo install mdbook
   ```

3. **Run locally**

   ```bash
   mdbook serve
   ```

4. **Open in browser**

   Navigate to `http://localhost:3000`

## 🔧 Development Workflow

### Building the Handbook

```bash
# Install dependencies
cargo install mdbook

# Build for production
mdbook build

# Serve locally for development
mdbook serve

# Deploy to GitHub Pages
mdbook build
cp -r book/* docs/
git add docs/
git commit -m "Update handbook"
git push origin main
```

### Content Management

- **Source files**: Located in `src/` directory
- **Images**: Stored in `src/images/`
- **Code examples**: Embedded in markdown files
- **Configuration**: `book.toml` file
