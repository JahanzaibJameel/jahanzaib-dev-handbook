<h1 id="introduction-to-react-native"><a class="header" href="#introduction-to-react-native">Introduction to React Native</a></h1> <p>React Native is a framework for building native mobile apps using JavaScript and React. It lets you write real, natively rendered mobile applications for iOS and Android - all from a single codebase.</p> <h2 id="-what-is-react-native"><a class="header" href="#-what-is-react-native">🎯 What is React Native?</a></h2> <p>React Native is:</p> <ul> <li><strong>Cross-platform</strong>: One codebase for iOS and Android</li> <li><strong>Native Performance</strong>: Uses actual native UI components</li> <li><strong>JavaScript-based</strong>: Use your web development skills</li> <li><strong>React-powered</strong>: Component-based architecture</li> <li><strong>Facebook-backed</strong>: Strong community and support</li> </ul> <h2 id="-why-react-native"><a class="header" href="#-why-react-native">🚀 Why React Native?</a></h2> <h3 id="advantages"><a class="header" href="#advantages">Advantages</a></h3> <ul> <li><strong>Single Codebase</strong>: Write once, deploy everywhere</li> <li><strong>Fast Development</strong>: Hot reloading, instant feedback</li> <li><strong>Native Feel</strong>: Real native UI components</li> <li><strong>Large Community</strong>: Extensive libraries and support</li> <li><strong>Cost Effective</strong>: Half the development cost of native apps</li> </ul> <h3 id="use-cases"><a class="header" href="#use-cases">Use Cases</a></h3> <ul> <li>Startups and MVPs</li> <li>Content-driven apps</li> <li>Social applications</li> <li>E-commerce mobile apps</li> <li>Business productivity apps</li> </ul> <h2 id="-core-concepts"><a class="header" href="#-core-concepts">🛠️ Core Concepts</a></h2> <h3 id="components"><a class="header" href="#components">Components</a></h3> <p>React Native uses the same component concepts as React:</p> <pre><code class="language-javascript">import React from 'react'; import { View, Text, StyleSheet } from 'react-native'; const WelcomeScreen = () =&gt; { return ( &lt;View style={styles.container}&gt; &lt;Text style={styles.title}&gt;Welcome to React Native!&lt;/Text&gt; &lt;/View&gt; ); }; const styles = StyleSheet.create({ container: { flex: 1, justifyContent: 'center', alignItems: 'center', }, title: { fontSize: 24, fontWeight: 'bold', }, }); export default WelcomeScreen; </code></pre> <h3 id="core-components"><a class="header" href="#core-components">Core Components</a></h3> <ul> <li><strong>View</strong>: Container component (like div in web)</li> <li><strong>Text</strong>: Text display component</li> <li><strong>Image</strong>: Image display component</li> <li><strong>ScrollView</strong>: Scrollable container</li> <li><strong>TextInput</strong>: User input field</li> <li><strong>TouchableOpacity</strong>: Pressable button</li> <li><strong>FlatList</strong>: Efficient list rendering</li> </ul> <h2 id="-architecture"><a class="header" href="#-architecture">🏗️ Architecture</a></h2> <h3 id="react-native-bridge"><a class="header" href="#react-native-bridge">React Native Bridge</a></h3> <p>React Native uses a bridge to communicate between JavaScript and native code:</p> <pre><code class="language-text">JavaScript Code → React Native Bridge → Native Modules → Native UI </code></pre> <h3 id="thread-model"><a class="header" href="#thread-model">Thread Model</a></h3> <ul> <li><strong>UI Thread</strong>: Renders native UI components</li> <li><strong>JavaScript Thread</strong>: Runs your JavaScript code</li> <li><strong>Shadow Thread</strong>: Calculates layouts</li> </ul> <h2 id="-setup-options"><a class="header" href="#-setup-options">📱 Setup Options</a></h2> <h3 id="1-expo-recommended-for-beginners"><a class="header" href="#1-expo-recommended-for-beginners">1. Expo (Recommended for Beginners)</a></h3> <p>Expo is a framework and platform for universal React applications.</p> <p><strong>Advantages:</strong></p> <ul> <li>No native code required</li> <li>Easy setup and deployment</li> <li>Built-in services (push notifications, OTA updates)</li> <li>Great for beginners</li> </ul> <p><strong>Installation:</strong></p> <pre><code class="language-bash">npm install -g expo-cli expo init my-app cd my-app npm start </code></pre> <h3 id="2-react-native-cli"><a class="header" href="#2-react-native-cli">2. React Native CLI</a></h3> <p>For more control and native module access.</p> <p><strong>Advantages:</strong></p> <ul> <li>Full control over native code</li> <li>Access to any native module</li> <li>Better for complex apps</li> </ul> <p><strong>Installation:</strong></p> <pre><code class="language-bash">npx react-native init MyApp cd MyApp npx react-native run-android # or run-ios </code></pre> <h2 id="-styling-in-react-native"><a class="header" href="#-styling-in-react-native">🎨 Styling in React Native</a></h2> <h3 id="stylesheet-api"><a class="header" href="#stylesheet-api">StyleSheet API</a></h3> <p>React Native uses JavaScript-based styling:</p> <pre><code class="language-javascript">import { StyleSheet } from 'react-native'; const styles = StyleSheet.create({ container: { flex: 1, backgroundColor: '#fff', padding: 20, }, button: { backgroundColor: '#007AFF', padding: 15, borderRadius: 8, alignItems: 'center', }, buttonText: { color: '#fff', fontSize: 16, fontWeight: 'bold', }, }); </code></pre> <h3 id="flexbox-layout"><a class="header" href="#flexbox-layout">Flexbox Layout</a></h3> <p>React Native uses Flexbox for layout (default direction is column):</p> <pre><code class="language-javascript">const styles = StyleSheet.create({ row: { flexDirection: 'row', // Horizontal layout justifyContent: 'center', // Horizontal alignment alignItems: 'center', // Vertical alignment }, column: { flexDirection: 'column', // Vertical layout (default) justifyContent: 'space-between', alignItems: 'stretch', }, }); </code></pre> <h2 id="-development-workflow"><a class="header" href="#-development-workflow">🔄 Development Workflow</a></h2> <h3 id="hot-reloading"><a class="header" href="#hot-reloading">Hot Reloading</a></h3> <ul> <li><strong>Fast Refresh</strong>: Instantly see changes</li> <li><strong>State Preservation</strong>: Component state maintained during reload</li> <li><strong>Error Recovery</strong>: Automatic recovery from errors</li> </ul> <h3 id="debugging-tools"><a class="header" href="#debugging-tools">Debugging Tools</a></h3> <ul> <li><strong>React Native Debugger</strong>: Comprehensive debugging</li> <li><strong>Flipper</strong>: Extensible mobile app debugger</li> <li><strong>Console Logs</strong>: Built-in logging</li> <li><strong>Remote Debugging</strong>: Chrome DevTools integration</li> </ul> <h2 id="-essential-libraries"><a class="header" href="#-essential-libraries">📦 Essential Libraries</a></h2> <h3 id="navigation"><a class="header" href="#navigation">Navigation</a></h3> <pre><code class="language-bash">npm install @react-navigation/native @react-navigation/stack npm install react-native-screens react-native-safe-area-context </code></pre> <h3 id="state-management"><a class="header" href="#state-management">State Management</a></h3> <pre><code class="language-bash">npm install @reduxjs/toolkit react-redux # or npm install @apollo/client graphql # for GraphQL </code></pre> <h3 id="ui-components"><a class="header" href="#ui-components">UI Components</a></h3> <pre><code class="language-bash">npm install react-native-elements npm install react-native-vector-icons </code></pre> <h3 id="utilities"><a class="header" href="#utilities">Utilities</a></h3> <pre><code class="language-bash">npm install react-native-async-storage npm install react-native-device-info npm install react-native-permissions </code></pre> <h2 id="-first-project-ideas"><a class="header" href="#-first-project-ideas">🎯 First Project Ideas</a></h2> <h3 id="beginner-projects"><a class="header" href="#beginner-projects">Beginner Projects</a></h3> <ol> <li><strong>Calculator App</strong>: Basic arithmetic operations</li> likes, and comments</li> </ol> <h2 id="-best-practices"><a class="header" href="#-best-practices">🚀 Best Practices</a></h2> <h3 id="component-structure"><a class="header" href="#component-structure">Component Structure</a></h3> <pre><code class="language-javascript">// Good component structure import React, { useState, useEffect } from 'react'; import { View, Text, StyleSheet } from 'react-native'; const MyComponent = ({ prop1, prop2 }) =&gt; { const [state, setState] = useState(null); useEffect(() =&gt; { // Side effects here }, []); return ( &lt;View style={styles.container}&gt; &lt;Text&gt;{prop1}&lt;/Text&gt; &lt;/View&gt; ); }; const styles = StyleSheet.create({ container: { // Styles here }, }); export default MyComponent; </code></pre> <h3 id="performance-tips"><a class="header" href="#performance-tips">Performance Tips</a></h3> <ul> <li>Use <code>FlatList</code> for long lists instead of <code>ScrollView</code> with <code>map</code></li> <li>Avoid unnecessary re-renders with <code>React.memo</code></li> <li>Use <code>useCallback</code> and <code>useMemo</code> for expensive operations</li> <li>Optimize images and use appropriate formats</li> <li>Test on real devices, not just simulators</li> </ul> <h2 id="-learning-path"><a class="header" href="#-learning-path">🎓 Learning Path</a></h2> <ol> <li><strong>Week
